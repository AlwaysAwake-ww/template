# 📌 템플릿 프로젝트


> "Spring boot 기반 템플릿 프로젝트"

---

## 🚀 프로젝트 소개

- ✅ 주요 목적: 신규 프로젝트에서 템플릿 프로젝트를 활용하여 개발환경 세팅의 효율성 증가
- ✅ 핵심 기능:
  - kakao login (OAuth)
  - Open Feign
  - Aop 로깅
  - Exception Handling
  - Health check
